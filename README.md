a forum website created by React, Next.js, MUI, TypeScript and deploy on Netlify.

## Quick Start

```bash
yarn install

yarn dev
```

then open browser and go to http://127.0.0.1:3000/

## Deploy

```bash
yarn build

yarn start
```
